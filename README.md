# veylan.dev

Personal landing page / profile hub for [veylan.dev](https://veylan.dev).

A single, self-contained static `index.html` — no build step, no dependencies.
Deployed on Vercel.

## Structure

- `index.html` — the entire site (HTML + inline CSS + a tiny year script)

## Editing

Open `index.html` and edit directly. Links live in the `.links` sections.
Push to `main` and Vercel redeploys automatically.
